Ordered List of HTML Tags

  1. <html>: The html tag is the root element of an HTML document. It encapsulates all other HTML elements and represents the beginning and end of the document.
  2. <head>: The head tag contains metadata about the HTML document, including its title, character set, and links to stylesheets and scripts.
  3. <title>: The title tag defines the title of the document, which is displayed on the browser tab and used as the page's name when bookmarked.
  4. <body>: The body tag contains the content of the HTML document, such as text, images, links, and other elements visible on the page.
  5. <h1> to <h6>: The h1 to h6 tags define HTML headings. <h1> is the highest level heading, and <h6> is the lowest.
  6. <p>: The p tag is used to define a paragraph of text. It is a block-level element that automatically adds space before and after the text.
  7. <a>: The a tag defines a hyperlink, which is used to link to other web pages, email addresses, or other content. The href attribute specifies the link's destination.
  8. <img>: The img tag is used to embed images in an HTML page. It requires the src attribute to define the image source and the alt attribute to provide alternative text.
  9. <ul> and <ol>: The ul tag defines an unordered list (bulleted), while the ol tag defines an ordered list (numbered). List items are placed within <li> tags.
  10. <div>: The div tag is a generic container for grouping and styling elements. It is a block-level element often used in web layouts.
